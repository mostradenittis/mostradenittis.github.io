How to page

Le strisce di codice sono scritte in corsivo.



Prima di iniziare confgurare git scrivendo nel terminale:

$ git config --global user.name "username"

$ git config --global user.email mail@example.com

Metti le credenziali del tuo account su github, non quello mostradenittis

Per clonare questa directory basta digitare nel terminale il comando:

$ git clone 'https://github.com/mostradenittis/mostradenittis.github.io.git'

Ed entrare nella cartella. Lì si potranno modificare i file. Per aggiornare all'ultima versione disponibile basta scrivere:

$ git pull

Una volta modificato il file può essere mandato al server digitando:

$ git add nomefile

oppure

$ git add --all

Fondamentale è il commit, per farlo bisogna scrivere:

$ git commit -m "messaggio di commit"

Usare un messaggio di commit che faccia capire cosa si è fatto.

Infine, per rendere le modifiche effettive, bisogna effettuare il push:

$ git push -u origin master

A questo punto basta inserire le credenziali dell'account di mostra de nittis
